Добавьте созданного бота в групповой чат с правами администратора.

Значение переменных с токеном бота (token), id группового чата (id_chat),
часа отправки (hour) и минуты отправки (minute) программа берет из переменных сред.
Переменные среды имеют следующий вид:
TOKEN значение - токен бота
CHAT_ID - id группового чата
QUESTION_HOUR - час отправки
QUESTION_MINUTE - минута отправки

В случае, если переменные среды не были определены, программа ищет их в файле конфигурации (config.ini). 
В файле config.ini нужно изменить следующие значения:

token = Ваш токен бота телеграмм
chat_id = Ваш id группы куда будет отправлен опрос
hour = час отправки
minute = минута отправки

id группового чата можно узнать с помощью сайта https://sean-bradley.medium.com/get-telegram-chat-id-80b575520659
Введите в поле HTTP API Token введите токен вашего бота, после чего в выданном коде найдите любое сообщение из вашего чата,
например:
 "chat": {
          "id": -1001913200647,  - id групповой беседы, который нам и нужен
          "title": "meow",      - название группового чата
          "type": "supergroup"
        },
        "date": 1684003890,
        "text": "ываыаываыв"   - текст сообщения
