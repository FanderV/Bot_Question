1. В 8 строке вместо token необходимо в кавычках указать токен телеграмм бота.

2. В 29 и 39 строках вместо id необходимо в кавычках указать id чата.
Получить id чата можно с помощью бота https://t.me/getmyid_bot
Нужно переслать сообщение из беседы, после чего бот отправит сообщение, где в строке "Forwarded from:" после двоеточия будет указан id. 

3. В 37 строке можно указать время отправки опроса ( по умолчанию 8:00).

4. Библитека telebot не является стандартной. Её нужно установить через pip install telebot.

5. Если вы используете Python версии 2.x, то requests нужно установить дополнительно через pip install requests. С версии 3.x и более библиотека стандартная и не требует дополнительных действий.

6. Бота необходимо добавить в беседу, стандартных прав достаточно.
